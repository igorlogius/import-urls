Import URLs from the clipboard or a text file as tabs or bookmarks Short Demo Video: https://github.com/igorlogius/open-tabs-from-file/assets/67047467/03f22538-c34c-4fd0-9ee0-1521c52fd0d6 Usage
  1. click the toolbar button to open the import UI
  2. choose the import source (clipboard or file)
  3. preview and edit the extracted urls
  4. open the URLs as tabs or add them into a bookmark folder
